# NH 포트폴리오 트래커

iPhone 15 Pro Max 최적화 PWA

## 배포 방법 (PC 불필요 — GitHub + Vercel 웹에서 전부 가능)

### 1단계. GitHub 업로드
1. github.com 로그인 → New repository → `nh-portfolio`
2. "uploading an existing file" 클릭
3. 이 ZIP의 파일들을 전부 드래그 업로드
4. Commit changes 클릭

### 2단계. Vercel 배포
1. vercel.com → GitHub으로 로그인
2. Add New Project → nh-portfolio 선택
3. Deploy 클릭 (설정 건드릴 필요 없음)
4. 1~2분 후 URL 생성됨

### 3단계. iPhone 홈화면 추가
1. Safari로 Vercel URL 접속
2. 하단 공유버튼(□↑) 탭
3. "홈 화면에 추가" 탭
4. 추가 → 앱처럼 실행됨 ✅
